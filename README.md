# Digitalisation Scan Documents

Cette application est une solution moderne permettant de transformer des documents physiques en fichiers numériques exploitables.  
Elle intègre des technologies de traitement d'image pour garantir une clarté optimale et faciliter l'archivage numérique.

## 🚀 Fonctionnalités

- 📸 Capture Intelligente : Détection automatique des bordures du document via la caméra.
- 🖼️ Traitement d'Image : Filtres d'amélioration (niveaux de gris, contraste élevé, suppression des ombres) pour une lisibilité parfaite.
- 🔍 OCR (Reconnaissance Optique de Caractères) : Extraction du texte à partir des images scannées.
- 📄 Export Multiformat : Génération de fichiers au format JPEG.
- 📂 Gestion Locale : Organisation des documents par catégories et stockage sécurisé.

## 🛠️ Technologies utilisées

- React
- TypeScript
- Vite
